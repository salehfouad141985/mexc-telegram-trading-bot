# 🤖 Shaabane Signals Auto-Trader Bot

بوت تداول آلي احترافي مبني باستخدام (Node.js) متصل بمنصة **MEXC** لنسخ وتنفيذ التوصيات تلقائيًا من القنوات العامة والخاصة على تطبيق **Telegram** (وتحديدًا قناة @shaabane_signals). 

يأتي البوت مجهزًا بلوحة تحكم ويب تفاعلية، نظام متقدم لإدارة المخاطر (وقف خسارة وجني أرباح متعدد الأهداف)، ووضعية المحاكاة (Dry Run) لاختبار وتأكيد الصفقات قبل المخاطرة بالأموال الحقيقية.

---

## 🌟 الميزات الرئيسية

- **⚡ التقاط سريع:** تقنية UserBot (MTProto) لسحب الإشارات في أقل من ثانية من أي قناة تليغرام دون الحاجة لأن يكون البوت مشرفاً (Admin).
- **🛡️ وضعية المحاكاة (Dry Run):** اختبر كفاءة الإشارات واستراتيجياتك بأموال وهمية قبل التفعيل على حسابك.
- **📈 إدارة ذكية للطلبات (MEXC):** ربط كامل مع منصة MEXC Spot API، لتقسيم الصفقات تلقائياً عبر (Take Profit 1, 2, 3, 4) وبناء وقف الخسارة.
- **🖥️ لوحة تحكم احترافية:** واجهة مستخدم رسومية (Dashboard) تعمل محلياً أو على خادم عبر متصفحك لمراقبة الأرباح وتاريخ التداولات وسجل النشاطات بالوقت الفعلي.
- **🧠 مفلتر الإشارات المتطور:** قراءة متقدمة باستخدام `Regex` لفهم سعر الدخول، وحالة الإشارة (SCORE > 7)، ونوع الصفقة للفلترة الذكية.
- **🗄️ قاعدة بيانات رشيقة:** حفظ كامل لتاريخ التداولات والتوصيات باستخدام `SQLite` لضمان السهولة وعدم تعقيد الإعدادات.

## 🛠️ الإعداد والتشغيل (Installation)

### 1. المتطلبات الأولية
- تثبيت [Node.js](https://nodejs.org/en) (الإصدار 18 فما فوق).
- حساب على [Telegram](https://telegram.org) للحصول على الـ `API_ID` و `API_HASH` الخاص بك من [my.telegram.org](https://my.telegram.org/).
- حساب على منصة [MEXC](https://www.mexc.com) ومفاتيح API لعمليات التداول.

### 2. تثبيت المشروع
قم بنسخ المستودع (Clone) ثم تثبيت الحزم المطلوبة:
```bash
git clone https://github.com/your-username/trading-bot-mexc.git
cd trading-bot-mexc
npm install
```

### 3. إعداد البيئة (Environment)
انسخ ملف `.env.example` إلى ملف جديد وقم بتسميته `.env`:
```bash
cp .env.example .env
```
افتح ملف `.env` وقم بتعبئة بياناتك (بيانات تليغرام، MEXC، وإعدادات إدارة رأس المال).

### 4. التشغيل لأول مرة
لبدء البوت:
```bash
npm start
```
* **ملاحظة:** في المرة الأولى سيدرج لك النظام تفاعلاً في شاشة التحكم لطلب **رقم الهاتف** ورمز التأكيد للولوج لحساب التيليغرام كـ (UserBot)، وبعدها سيتم حفظ الجلسة محلياً لعدم إزعاجك مجدداً!

### 5. متابعة لوحة التحكم
افتح الرابط التالي في متصفحك المفضل:
👉 **[http://localhost:3030](http://localhost:3030)**

## ⚙️ إعدادات المتغيرات المهمة (.env)

- `DRY_RUN=true`: في الوضع `true` سيبدأ النظام بمحاكاة الأرباح وتسجيل الصفقات بدون الاتصال بحساب MEXC. اجعلها `false` للمتاجرة الحقيقية.
- `TRADE_AMOUNT_USDT=10`: مبلغ التداول بالدولار الذي تريد استثماره في **الصفقة الواحدة**.
- `MIN_SCORE=7`: البوت سيرفض الدخول في أي توصية يكون التقييم فيها أقل من الرقم المعطى (مثل 7/10).

---

## 🔒 الملاحظات الأمنية

- احرص على إبقاء ملف الـ `.env` آمناً ولا ترفعه للإنترنت أبدًا.
- قم بتفعيل صلاحيات القراءة/التداول (`Read` and `Trade`) فقط في لوحة MEXC API، وتأكد من **عدم إعطاء صلاحيات السحب (Withdrawals)**.
